Installation

You can install graphql from RubyGems by adding to your application’s Gemfile:

1# Gemfile
2gem "graphql"

Then, running bundle install:

1$ bundle install

Getting Started

Building a GraphQL server goes like this:

  • Define some types
  • Connect them to a schema
  • Execute queries with your schema

Declare types

Types describe objects in your application and form the basis for GraphQL’s type system.

 1PostType = GraphQL::ObjectType.define do
 2  name "Post"
 3  description "A blog post"
 4  # `!` marks a field as "non-null"
 5  field :id, !types.ID
 6  field :title, !types.String
 7  field :body, !types.String
 8  field :comments, types[!CommentType]
 9end
10
11CommentType = GraphQL::ObjectType.define do
12  name "Comment"
13  field :id, !types.ID
14  field :body, !types.String
15  field :created_at, !types.String
16end

Build a Schema

Before building a schema, you have to define an entry point to your system, the “query root”:

 1QueryType = GraphQL::ObjectType.define do
 2  name "Query"
 3  description "The query root of this schema"
 4
 5  field :post do
 6    type PostType
 7    argument :id, !types.ID
 8    description "Find a Post by ID"
 9    resolve ->(obj, args, ctx) { Post.find(args["id"]) }
10  end
11end

Then, build a schema with QueryType as the query entry point:

1Schema = GraphQL::Schema.define do
2  query QueryType
3end

This schema is ready to serve GraphQL queries! See “Configuration Options” for all the schema options.

Execute queries

You can execute queries from a query string:

 1query_string = "
 2{
 3  post(id: 1) {
 4    id
 5    title
 6  }
 7}"
 8result_hash = Schema.execute(query_string)
 9# {
10#   "data" => {
11#     "post" => {
12#        "id" => 1,
13#        "title" => "GraphQL is nice"
14#     }
15#   }
16# }

See “Executing Queries” for more information about running queries on your schema.

Use with Relay

If you’re building a backend for Relay, you’ll need:

  • A JSON dump of the schema, which you can get by sending GraphQL::Introspection::INTROSPECTION_QUERY
  • Relay-specific helpers for GraphQL, see the GraphQL::Relay guides.

Use with Apollo Client

Apollo Client is a full featured, simple to use GraphQL client with convenient integrations for popular view layers. You don’t need to do anything special to connect Apollo Client to a graphql-ruby server.
