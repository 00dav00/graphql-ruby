Class: GraphQL::Language::Nodes::AbstractNode < Object AbstractNode is the base class for all nodes in a GraphQL AST. It provides some APIs for working with ASTs: - children returns all AST nodes attached to this one. Used for tree traversal. - scalars returns all scalar (Ruby) values attached to this one.
